Steps to run this application:

01. Download BricksOrderingSystem.zip file
02. Extract the zip file.
03. Import both bricks-ordering-system and bricks-ordering-client as existing maven projects.
04. Run BricksOrderingSystemApplication.java file to start bricks-ordering-system (Spring boot application) which is the REST service here.
05. In order to test bricks-ordering-client, run BricksOrderingClientTest.java as JUnit test.
